📕 **一站式小红书内容创作工具**

**爆款文案生成** · **热点词库** · **违禁词检测** · **13 种风格封面卡片渲染**

---

## 1. 技能定位

### 核心能力

| 功能模块 | 描述 | 输出 |
|----------|------|------|
| **爆款帖子生成** | 基于主题生成完整笔记方案 | 标题 + 正文 + 标签 + 互动 + CTA |
| **热点词库生成** | SEO 优化词库，提升曝光率 | 热搜词 + 长尾词 + 标题改造 |
| **违禁词检测** | 三级风险扫描，避免违规 | 风险等级 + 替换建议 |
| **封面卡片渲染** | 13 种风格，生成专业图片 | 封面 + 正文卡片（1080x1440） |

### 适用场景

- ✅ 自媒体运营者批量创作笔记
- ✅ 品牌方制作产品种草内容
- ✅ 个人博主提升内容质量
- ✅ 营销团队标准化内容产出

### 5.1 post - 生成爆款帖子

```bash
python3 xiaohongshu.py post <主题>
```

**输出结构：**

```
==================================================
📕 小红书爆款文案
==================================================

📌 标题：[2 个 emoji + 数据化结论 + 获得感]

📝 正文：
[场景共鸣]
[模块化方法论]
[低门槛呼吁]

🏷️ 话题标签：[5 个精准标签]

💬 互动问题：
  • 经验：[经验共享型]
  • 观点：[观点对抗型]
  • 资源：[资源交换型]

📣 CTA 策略：
  • 硬转化：[直接引导]
  • 软引导：[关注/收藏]
==================================================
```

**示例：**

```bash
python3 xiaohongshu.py post 学渣逆袭
```

### 5.4 render - 渲染封面卡片

```bash
python3 scripts/render_xhs.py <markdown 文件> [选项]
```

**命令行选项：**

```
必选参数:
  markdown_file         Markdown 文件路径

可选参数:
  -t, --theme           排版主题（默认：memphis）
  -m, --mode            分页模式（默认：separator）
  -o, --output-dir      输出目录（默认：当前目录）
  -w, --width           图片宽度（默认：1080）
  --height              图片高度（默认：1440）
```

**主题选项（13 种）：**

```
memphis         孟菲斯（好物推荐/种草）
acid            酸性设计（技术编程/数码）
vaporwave       蒸汽波（生活美食/旅行）
minimalist      极简主义（知识干货/职场）
glitch          故障艺术（科技/游戏）
chinese         中国风（传统文化/国风）
glassmorphism   毛玻璃（美妆穿搭/护肤）
bauhaus         包豪斯（设计/建筑/艺术）
pop-art         波普艺术（时尚/潮流）
art-nouveau     新艺术运动（艺术/花卉）
collage         拼贴艺术（创意/手作）
new-ugly        新丑风格（个性表达/实验）
brutalism       粗野主义（极简/工业风）
```

**分页模式：**

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| `separator` | 按 `---` 手动分页 | 精确控制每页内容 |
| `auto-fit` | 自动缩放文字填满 | 单页内容 |
| `auto-split` | 按高度自动切分 | 长文自动分页 |
| `dynamic` | 动态调整图片高度 | 灵活适应内容 |

**示例：**

```bash
# 孟菲斯风格
python3 scripts/render_xhs.py examples/tool-recommendation.md -t memphis -o ./output

# 酸性设计风格
python3 scripts/render_xhs.py examples/tool-recommendation.md -t acid -o ./output

# 一键测试所有风格
for theme in memphis acid vaporwave minimalist glitch chinese glassmorphism bauhaus pop-art art-nouveau collage new-ugly brutalism; do
  python3 scripts/render_xhs.py examples/tool-recommendation.md -t $theme -o "./output/$theme"
done
```

## 6. Markdown 格式规范

### 完整模板

```markdown
---
emoji: "🚀"
title: "5 个效率神器"
subtitle: "打工人必备，效率翻倍"
---

# 📝 神器一：Notion

> 全能型笔记工具，支持数据库、看板、日历

## 核心功能

- 数据库视图（表格/看板/日历）
- 双向链接 + 知识图谱
- AI 智能助手集成

## 使用场景

适合做知识库、项目管理、个人 Wiki。

**价格**：$39/月起

---

# ⚡ 神器二：Raycast

快捷键呼出的效率工具，替代 Spotlight。

## 推荐理由

- 启动速度快（<100ms）
- 插件生态丰富
- 原生 Mac 体验

**价格**：免费版可用
```

### 字段说明

| 字段 | 类型 | 必需 | 说明 | 示例 |
|------|------|------|------|------|
| `emoji` | string | ✅ | 封面装饰 Emoji | `"🚀"` |
| `title` | string | ✅ | 封面大标题（≤15 字） | `"5 个效率神器"` |
| `subtitle` | string | ✅ | 封面副标题（≤15 字） | `"打工人必备"` |

### 分页控制

使用 `---` 在内容中创建分页：

```markdown
# 第一页内容

...

---

# 第二页内容

...
```

## 7. 质量标准

### 文案质量标准

| 检查项 | 标准 | 验证方法 |
|--------|------|----------|
| 标题长度 | ≤20 字（含 emoji） | 数字符 |
| 标签数量 | 5 个（基础 2+ 长尾 2+ 热点 1） | 数个数 |
| 互动问题 | 3 种类型各 1 条 | 检查完整性 |
| CTA 策略 | 1 硬 1 软 | 检查组合 |
| 违禁词风险 | 🟢绿色 或 🟡黄色（已修改） | check 命令验证 |

### 图片质量标准

| 检查项 | 标准 | 验证方法 |
|--------|------|----------|
| 图片尺寸 | 1080x1440px | 查看属性 |
| 卡片数量 | 3-6 张（根据内容） | 数文件 |
| 文字清晰 | 无溢出/截断 | 目视检查 |
| 主题匹配 | 符合内容调性 | 主观判断 |
| Emoji 显示 | 彩色 Noto Emoji | 目视检查 |

## 9. 常见问题处理

### 文案生成问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 标题太长 | 超过 20 字 | 精简表述，去掉冗余词 |
| 标签不精准 | 太泛或太偏 | 参考 keywords 命令输出 |
| 文案太生硬 | 缺乏共鸣 | 增加场景描述和 Emoji |

### 图片渲染问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 渲染失败 | Markdown 格式错误 | 检查 YAML 头部，确保 `---` 成对 |
| 图片模糊 | DPR 太低 | 加 `--dpr 3` 参数（如支持） |
| 文字溢出 | 字数太多 | 改用 `auto-split` 或精简内容 |
| 主题不好看 | 主题不匹配 | 换主题，参考主题选择指南 |
| Emoji 黑白 | 字体加载失败 | 检查网络，确保 Google Fonts 可访问 |

### 违禁词检测问题

| 风险等级 | 含义 | 处理建议 |
|----------|------|----------|
| 🔴 红色 | 发现严重违规词 | 必须修改后发布 |
| 🟡 黄色 | 发现潜在风险词 | 建议修改 |
| 🟢 绿色 | 未发现明显问题 | 可安全发布 |

## 10. 最佳实践

### ✅ 推荐做法

1. **文案生成后先检测**：用 `check` 命令扫描违禁词
2. **结合热点词库**：用 `keywords` 优化标题和标签
3. **选择合适的渲染风格**：参考主题选择指南
4. **测试多种风格**：一键生成 13 种风格对比效果
5. **保留 Markdown 源文件**：方便后续修改重渲染

### ❌ 避免做法

1. **直接使用未检测的文案**：可能违规限流
2. **标题超过 20 字**：影响点击率
3. **标签过多或过少**：固定 5 个最佳
4. **忽略互动设计**：降低笔记活跃度
5. **图片内容太满**：留白提升高级感

## 11. 完整工作流示例

### 案例：AI 工具推荐笔记

```bash
# 步骤 1：生成爆款文案
python3 xiaohongshu.py post AI 工具

# 步骤 2：生成热点词库（优化标签）
python3 xiaohongshu.py keywords AI 工具

# 步骤 3：违禁词检测（确保合规）
python3 xiaohongshu.py check "生成的文案内容"

# 步骤 4：准备渲染用 Markdown
cat > ai-tools.md << 'EOF'
---
emoji: "🤖"
title: "3 个 AI 写作神器"
subtitle: "打工人必备，效率翻倍"
---

# 📝 Jasper

> 全能型 AI 写作助手，模板超丰富

## 核心功能

- 博客文章生成
- 邮件文案优化
- 广告创意写作

---

# ⚡ Copy.ai

专注营销文案，一键生成产品描述。

## 价格

💰 免费版可用
EOF

# 步骤 5：渲染封面卡片（测试 3 种风格）
python3 scripts/render_xhs.py ai-tools.md -t memphis -o ./output/memphis
python3 scripts/render_xhs.py ai-tools.md -t acid -o ./output/acid
python3 scripts/render_xhs.py ai-tools.md -t minimalist -o ./output/minimalist

# 步骤 6：查看结果，选择最佳风格
ls -la ./output/*/

# 步骤 7：交付完整素材包
# - 文案（步骤 1 输出）
# - 图片（步骤 5 输出）
# - Markdown 源文件（ai-tools.md）
```
