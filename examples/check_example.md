# 违禁词检测示例

## 示例 1: 高风险文本

### 输入
```bash
python3 xiaohongshu.py check "这款面膜全网销量第一！医院皮肤科都在用，7 天淡化所有皱纹，添加的 xxx 成分能修复受损细胞，私信我送独家护肤秘籍"
```

### 输出
```
==================================================
🛡️ 违禁词检测报告
==================================================

📄 检测文本：这款面膜全网销量第一！医院皮肤科都在用...

📊 综合风险等级：🔴 红色（发现 5 处问题）

🚨【违规词 1】"全网销量第一"
   类型：绝对化用语（违反《广告法》第 9 条）
   建议改为："累计热销超 10 万盒"

🚨【违规词 2】"医院皮肤科都在用"
   类型：医疗关联暗示（违反平台医疗美容类目规定）
   建议改为："实验室实测数据表明"

🚨【违规词 3】"7 天淡化所有皱纹"
   类型：虚假效果承诺
   建议改为："28 天可见肤质改善"

🚨【违规词 4】"修复受损细胞"
   类型：医疗功能宣称
   建议改为："帮助强韧肌肤屏障"

🚨【违规词 5】"私信我送..."
   类型：站外引流风险
   建议改为："评论区抽奖赠送"
==================================================
```

## 违禁词分类说明

### 🔴 绝对化用语
- **禁用词**：最、第一、国家级、全网、顶级、史上、永久、万能、唯一、极致、首选
- **违反条款**：《广告法》第 9 条
- **替代方案**：个人觉得、实测感受、累计热销超 X 万、口碑爆款

### 🔴 医疗宣称
- **禁用词**：根治、疗效、修复细胞、治疗、药方、医生推荐、医院同款、处方、疗程、皮肤科
- **违反条款**：平台医疗美容类目规定
- **替代方案**：帮助改善、缓解、使用效果、实测反馈、实验室数据

### 🔴 效果保证
- **禁用词**：7 天、月瘦 20 斤、立即见效、100% 有效、无效退款、永不反弹、三天见效
- **违反条款**：《广告法》第 28 条（虚假广告）
- **替代方案**：28 天可见改善、多数人反馈、持续使用可见效果

### 🔴 引流信息
- **禁用词**：加微信、二维码、私信我、外链、公众号、淘宝店、微店、联系方式
- **违反条款**：平台社区规范
- **替代方案**：评论区交流、评论区留言、主页查看

### 🔴 封建迷信
- **禁用词**：风水、算命、转运、招财、辟邪、改运、开光、保佑
- **违反条款**：平台内容安全规范
- **替代方案**：删除相关表述
