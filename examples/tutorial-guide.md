# 🎯 找准定位

> 选你擅长 + 热爱的领域

## 三步定位法

1. **擅长什么** - 列出你的技能/经验
2. **热爱什么** - 能持续输出的内容
3. **市场要什么** - 看对标账号数据

## 差异化策略

- 找细分领域
- 打造个人特色
- 坚持统一风格

# ⏰ 发布技巧

> 黄金时间 + 吸睛标题

## 最佳发布时间

- 🌅 早 8:00 - 通勤时段
- ☀️ 午 12:00 - 午休时段
- 🌆 晚 18:00 - 下班时段

## 标题公式

- 数字 + 结果：「7 天涨粉 1000」
- 疑问句：「为什么你没人看？」
- 对比：「从 0 到 1 我做对了什么」

# 📊 数据分析

## 关键指标

- **小眼睛** - 浏览量
- **互动率** - 赞藏评/浏览
- **完读率** - 内容吸引力

## 优化方向

1. 浏览低 → 优化标题封面
2. 互动低 → 增加互动引导
3. 涨粉慢 → 强化人设定位
